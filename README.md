# `heat-tree`: A JavaScript package for custom, dynamic phylogenetic tree visualizations

Under development.

Features:
  - Reads tree data from Newick files
  - Reads data associated with tips/nodes in tree as a TSV file.
  - Zooming and panning
  - Nodes can be collapsed and the layout of the rest of tree responds
  - Nodes/tips have sizes and colors that correspond to continuous variables
  - Nodes/tips have shapes colors that correspond to categorical variables
  - Branches have thicknesses and colors that correspond to continuous variables
  - Which variable use to set node/tip/branch colors/sizes can be changed by embedded controls
